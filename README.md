# OpenRisk AI

OpenRisk AI is an autonomous Web3 attack surface discovery platform that helps security researchers identify protocol trust assumptions, attack surfaces, and vulnerability hypotheses.

## Features

- GitHub repository ingestion
- Solidity contract discovery
- Attack surface mapping
- Trust assumption analysis
- Vulnerability hypothesis generation
- Security report generation

## Tech Stack

- FastAPI
- Python
- ChromaDB
- LLM Agents
- GitPython
- Next.js
